## Usage
* curl is a hard dependency, it must be located in `$PATH`
* must be macOS or Linux.

```shell
hikaru add <room_id>      # add idol to crontab
hikaru crontab            # check live idol's status
hikaru run <room_id>      # make idol shine!
```
